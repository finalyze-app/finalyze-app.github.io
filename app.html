Rules

Auto-categorization rules

Run before built-in rules · drag to reorder

Match merchant text with a regular expression and assign a category. Rules are checked top-to-bottom, before the built-in keyword rules. Merchant overrides still win.

Groups

Roll up categories for charts & budgets

Combine categories like Dining + Groceries into “Food”. Use the category chart’s Groups view to see rollups. Each category can belong to one group.
