Categories

Changing categories

When you change a category in the transaction ledger or uncategorized queue, choose the default behavior. Bulk recategorize in the ledger always updates every selected merchant.

Preferences

Manage categories

Add your own categories and recolor any category. Built-in categories can be recolored; custom ones can be removed.

Rules

Auto-categorization rules Pro

Run before built-in rules · drag to reorder

Match merchant text with a regular expression and assign a category. Rules are checked top-to-bottom, before the built-in keyword rules. Merchant overrides still win.

Rules

Subscription rules Pro

Flag recurring by keyword - any amount

Match merchant text with a keyword or regular expression to always treat it as recurring/subscription in the Recurring widget - even when the charge amount changes each time (e.g. usage-based bills).

Rules

Auto-merge rules Pro

Fold matching merchants into one name

Automatically combine differently-named merchants into a single canonical name on every import - e.g. match AMZN|AMAZON and merge into “Amazon”. Applies to all past and future transactions.

Overview

Custom cards Pro

Shown in Spending overview · totalled for the selected period

Build a KPI card from criteria on your transactions (e.g. Category is Health AND Description contains “cpap”). The card appears in the Spending overview and totals matching transactions for whatever period is selected (This Month, Last Year, a custom range…), with the period appended to its name.

Groups

Category groups

Roll up categories for charts & budgets

Combine categories like Dining + Groceries into “Food”. Use the category chart’s Groups view to see rollups. Each category can belong to one group.

Budgets

Monthly budgets

Alerts when the current month exceeds the limit

Set limits on groups (combined member spend) or individual categories not in a group.

Accounts

Accounts & storage

Imports can be assigned to an account

Track multiple cards/accounts separately. Use the account filter in the header to scope the dashboard, or "All accounts" for household totals.

Merchants

Merge merchants

Tick two or more, then merge

Group differently-named merchants into one. The merge is remembered, so future imports with any of the old names roll up automatically.

Cardmembers

Merge cardmembers

Tick two or more, then merge

Combine duplicate cardholder names (e.g. “JOHN DOE” and “JOHN D”). Merges are remembered and applied to charts, filters, and the cardmember breakdown.
